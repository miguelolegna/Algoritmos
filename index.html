Algoritmos

Um algoritmo é uma sequência finita e ordenada de passos claros e precisos que visam resolver um problema ou realizar uma tarefa específica. Esses passos devem ser suficientemente simples para serem executados por um agente humano ou implementados por um sistema computacional.

Características Principais

  • Finitude: Os algoritmos devem ser compostos por um número finito de etapas. Isso significa que, em algum ponto, o algoritmo deve terminar, seja encontrando a solução para o problema ou reconhecendo que a solução não é possível.
  • Precisão: Cada iteração do algoritmo é claramente definida, indicando se o elemento está na metade superior ou inferior da lista.
  • Ordenação:< Os passos do algoritmo são executados em uma sequência ordenada, seguindo a lógica de dividir a lista pela metade.
  • Efetividade: Cada passo é viável, pois a lista é reduzida pela metade a cada iteração, tornando o processo eficiente mesmo em listas grandes.

Tipos de Algoritmos

Ordenação e Busca:
  • Descrição: Algoritmos de ordenação organizam elementos em uma determinada ordem, como crescente ou decrescente. Algoritmos de busca, por outro lado, procuram por um elemento específico em uma coleção de dados.
  • Exemplo: Algoritmo de Ordenação Bolha, Algoritmo de Busca Binária.
    • Dividir e Conquistar:
  • Descrição: Estratégia que divide um problema complexo em subproblemas menores, resolve cada subproblema e combina as soluções para obter a solução do problema original.
  • Exemplo: Algoritmo Merge Sort, Algoritmo Quick Sort.
    • Gulosos (Greedy):
  • Descrição: Algoritmos gulosos tomam decisões locais que parecem ser as melhores em cada etapa, na esperança de chegar à melhor solução global.
  • Exemplo: Algoritmo da Mochila Fracionária, Algoritmo de Kruskal para árvores geradoras mínimas.
    • Dinâmica de Programação:
  • Descrição: Aborda problemas dividindo-os em subproblemas sobrepostos, resolvendo cada subproblema apenas uma vez e armazenando suas soluções para evitar recálculos.
  • Exemplo: Algoritmo de Floyd-Warshall para o caminho mais curto, Algoritmo de Fibonacci usando programação dinâmica.
    • Exemplo de Aplicação: Algoritmo de Ordenação Rápida (Quick Sort)
  • Descrição: O algoritmo Quick Sort segue a abordagem "dividir para conquistar". Ele seleciona um elemento, chamado de pivô, e divide a lista em dois subconjuntos - elementos menores que o pivô e elementos maiores que o pivô. Em seguida, repete esse processo nos subconjuntos.
